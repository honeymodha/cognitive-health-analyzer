# Cognitive-Health-Analyzer

A simple web-based tool that analyzes your daily habits and provides a cognitive health score.

🚀 Features
Sleep tracking  
Screen time analysis  
Reading habit evaluation  
Personalized recommendations  
Interactive UI using sliders  

🛠️ Technologies Used  
HTML  
CSS  
JavaScript
